# Candidate documentation inputs

Put documentation excerpts here before running the L0 audit.

Expected layout:

```text
candidate_docs/<dataset_id>/README.md
candidate_docs/<dataset_id>/paper_notes.txt
candidate_docs/<dataset_id>/dataset_page.txt
```

The `dataset_id` must match an entry in `candidate_datasets.yaml`.

Do not commit full papers if copyright is unclear. Prefer short notes, copied public README text, dataset metadata, and your own extracted timing notes.
